1. Download Git from git-scm.com/downloads
2. Install git
3. Verify that git is installed by executing: git on the console
4. Go to github.com and create account. If you already have account login.

git@github.com:ocularminds/bootcamp.git
git commands
1. git clone - copy code locally
2. git pull - update code locally/download update from remote server
3. git push - save the code to remote server/upload updates to remote server
4. git add - stages the files to be saved
5. git commit - saved the code locally for remote upload
